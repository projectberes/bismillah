Pertanyaan yang Sering Ditanyakan

Masih ragu? Ini jawaban untuk pertanyaan yang paling umum.

Berapa lama waktu pengerjaan project?
+

Tergantung kompleksitas project. Untuk tugas sederhana bisa selesai dalam 1-2 hari. Website multi-halaman biasanya 3-7 hari. Project kompleks bisa 1-2 minggu. Deadline selalu kita diskusikan di awal agar sesuai kebutuhanmu.

Apakah ada garansi revisi?
+

Ya! Setiap paket sudah termasuk revisi gratis. Paket Starter 1x revisi, Professional 3x revisi, Enterprise unlimited revisi. Revisi meliputi perubahan tampilan, penambahan konten kecil, atau penyesuaian fungsionalitas sesuai brief awal.

Bagaimana sistem pembayarannya?
+

Untuk project di bawah Rp500K, pembayaran bisa penuh di awal. Untuk project besar, sistem DP 50% di awal dan 50% setelah selesai. Kami menerima transfer bank, GoPay, OVO, DANA, dan QRIS.

Apakah source code diberikan ke saya?
+

Tentu! Semua file source code, aset, dan dokumentasi akan diserahkan sepenuhnya kepada klien setelah pembayaran lunas. Kamu bebas menggunakan, memodifikasi, atau mengembangkan lebih lanjut.

Apakah bisa konsultasi dulu sebelum pesan?
+

Sangat bisa! Konsultasi awal 100% gratis. Ceritakan kebutuhanmu via WhatsApp, kita diskusi bersama tentang scope, estimasi harga, dan timeline. Tidak ada kewajiban untuk langsung pesan.

Apakah project kampus/skripsi juga bisa dikerjakan?
+

Bisa banget! Banyak klien kami adalah mahasiswa yang butuh bantuan project akademik seperti website tugas akhir, sistem informasi skripsi, atau program untuk mata kuliah. Semua dikerjakan profesional dan sesuai kebutuhan akademik.
